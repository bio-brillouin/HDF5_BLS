HDF5 File Visualizer

This tool helps you explore Brillouin Light Scattering (BLS) data stored in a Hierarchical Data Format (HDF5) file. HDF5 organizes data in a human- and machine-readable hierarchical structure using groups and datasets.

The standard structure for a single measure is a `Brillouin` group containing a `Measure` group, which itself contains a `Measure` dataset. Groups are used for organization and storing metadata, while datasets hold the actual data.

Drag and drop an HDF5 file here or click to select a file.
